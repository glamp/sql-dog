What is this?

This is a SQL client that runs entirely in the browser! I built it out of frustration with existing SQL IDEs for OSX. My biggest issue is that most SQL clients are built for database administrators, NOT database USERS. Think about it...who writes the most queries? Not developers but analysts!

What does it do?

  • Run queries in the broswer
  • Predictive type everywhere
  • Saves your queries using git
  • Save results in the browser, or export to Excel/CSV
  • Help with executing long running queries
  • Customizeable macros / snippets
  • Customizeable keyboard shortcuts
  • Query templating
